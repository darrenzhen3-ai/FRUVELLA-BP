# Roti Gulung Buah - Ice Cream Website Template

This is a responsive website template for an ice cream brand, designed to match the clean and playful UI from the provided Figma design video.

## Features

- **Responsive Design**: Optimized for desktop, laptop, tablet, and mobile devices
- **Smooth Animations**: Includes scroll animations, hover effects, and playful transitions
- **Modern UI**: Clean design with gradient backgrounds and card-based layouts
- **Interactive Elements**: Hamburger menu for mobile, smooth scrolling navigation
- **Playful Animations**: Floating elements, parallax effects, and fade-in animations

## File Structure

```
rotigulungbuah/
├── index.html          # Main HTML file
├── css/
│   └── style.css       # Main stylesheet
├── js/
│   └── script.js       # JavaScript for animations and interactions
└── README.md           # This file
```

## How to Use

1. Ensure you have XAMPP installed and running
2. Place this folder in your `htdocs` directory (e.g., `C:\xampp\htdocs\rotigulungbuah`)
3. Start Apache server in XAMPP control panel
4. Open your browser and navigate to `http://localhost/rotigulungbuah`

## Customization

- **Colors**: Edit the CSS variables in `style.css` to match your brand colors
- **Content**: Update text and images in `index.html`
- **Images**: Replace placeholder images with your actual product images
- **Fonts**: Modify the Google Fonts link in `index.html` if needed
